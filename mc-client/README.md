- Mod List

### MODS

- No fire overlay 

## Cosmetics
- Wings
- Hat
- Capes
- Blaze Effect
- (with settings, color etc)
## How to install:
- Download the release .zip folder 
- Put it in %appdata%/minecraft/versions/StuluClient
- Unzip
- Create minecraft profile
- Execute the profile
